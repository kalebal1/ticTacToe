This is a simple tic-tac-toe game made with guidance from a Hyperskill project.
Once completed, the user can play against the computer at various difficulty levels. 


The field is set up as a 2-dimensional array with the coordinates as follows:
(1, 3) (2, 3) (3, 3)
(1, 2) (2, 2) (3, 2)
(1, 1) (2, 1) (3, 1)
